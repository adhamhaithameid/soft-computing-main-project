# Dataset Decision

## Decision
Use **Epileptic Seizure Recognition**.

## Why
- Tabular and easier than image-heavy options.
- Enough dimensionality to justify PCA/LDA/SVD and feature selection.
- Supports both binary and multiclass exercises.
- Good fit for full method coverage from lectures.

If Kaggle access is unavailable, fetch from UCI through `ucimlrepo`.
