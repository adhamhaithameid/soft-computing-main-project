# Master Plan

## Objective
Build a full-course soft computing project using one dataset and covering preprocessing, feature reduction, feature selection, classification, GA, evaluation, graphing, and paper writing.

## Chosen dataset
Epileptic Seizure Recognition.

## Preferred execution platform
Google Colab (with local run support).

## Tracks
- Binary classification (seizure vs non-seizure)
- Multiclass classification

## Method coverage
- Preprocessing and statistical analysis
- PCA, LDA (reduction), SVD
- Filter/wrapper/embedded feature selection
- Genetic Algorithm feature subset optimization
- KNN, SVM, Decision Tree, Logistic Regression, LDA classifier, ANN/MLP
- K-fold CV, confusion matrix, ROC, and full metric suite
