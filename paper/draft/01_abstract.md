# Abstract

This project investigates soft computing techniques for epileptic seizure recognition using two prediction tracks: binary seizure detection and multiclass classification. We apply preprocessing, feature reduction (PCA/LDA/SVD), feature selection (filter, wrapper, embedded, and genetic algorithm), and multiple classifiers (KNN, SVM, Decision Tree, Logistic Regression, LDA classifier, and ANN/MLP). Results are evaluated with 5-fold cross-validation, confusion matrices, and standard classification metrics to compare performance across methods.
